Установщик (exe) сделан через InstallForge
MSI сделан через MSI Wrapper

Логика скрипта:
Установщик закидывает скрипт и батник в папку, выбранную юзером (по умолчанию путь в установщике стандартный к папке DLC)
Установщик запускает батник, батник запускает скрипт
Скрипт:
Ищется последний релиз а публичном гитхабе (кэтса), из названия берётся версия
Скрипт устанавливает рабочую директорию для себя
В этой директории удаляет папки с названием, включающим слово Tournament
Потом качает релиз
Распаковывает
Запускает ui_check
Удаляет архив
Удаляет сам себя
Батник тоже самоудаляется

Скрипт может не работать из-за впн (блочится API гитхаба)
Если возникла ошибка, то юзер об этом не узнает (надо доработать)
При запуске MSI может вылезать окно Windows Smart Screen
